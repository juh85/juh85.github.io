Sobre mim

Formada em Análise e Desenvolvimento de Sistemas, sou uma desenvolvedora curiosa, determinada e atenta aos detalhes.

Minha trajetória começou na moda esportiva, desenvolvendo moldes para peças piloto — experiência que me ensinou a trabalhar em equipe, pensar estrategicamente e buscar soluções criativas com foco em eficiência.

A transição para o desenvolvimento web surgiu quando percebi o quanto gosto de ver ideias ganhando forma. Assim como um molde se transforma em roupa, o código se transforma em uma aplicação funcional.

Hoje sigo aprimorando minhas habilidades e criando projetos que reforçam organização, clareza e boas práticas de desenvolvimento.

Previsão de clima

Este projeto é uma aplicação web de previsão do tempo, desenvolvida com foco no aprendizado de consumo de APIs com organização de código e boas práticas no front-end e back-end. Permite buscar a previsão por cidade, exibe dados climáticos, altera a imagem de fundo conforme o clima e conta com autocomplete para facilitar a busca.
